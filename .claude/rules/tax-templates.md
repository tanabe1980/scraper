# 確定申告テンプレートのバージョン管理ルール

## 基準年度

- **2025年(R07)** を基準年度とする。
- `taxTemplates.ts` の `BASELINE_TAX_YEAR` 定数で管理する。

## テンプレート取得の挙動

`getTemplateForTaxYear(formCode, taxYear)` の動作:

| 条件 | 返すテンプレート |
|---|---|
| `TEMPLATE_MAP` に対象年度のエントリがある | そのエントリ |
| エントリがなく、対象年度 > 基準年度 | 基準年度(2025年)のテンプレート |
| エントリがなく、対象年度 < 基準年度 | `null`(未対応) |

## 新年度の対応手順

1. 国税庁が書式を改訂した年度は `TEMPLATE_MAP` に新エントリを追加する。
2. 追加したエントリはその年以降の基準となる(`BASELINE_TAX_YEAR` は変更しない)。
3. PDF・フィールドマップファイルを `/templates/<year>/` に配置する。
4. フィールドマップ(`*.json`)を更新し、フォームページのレイアウトを検証する。

## ページ実装のルール

- 新年度の青色決算書・申告書Bページを実装する際は、2025年のページ実装を基準に開始する。
- 書式変更が発生した年度のページは、その差分のみを追加実装し、それ以前の年度は影響しない。
- フォームコードは以下を維持する:
  - `blue`: 青色申告決算書
  - `kakutei_b`: 申告書B
  - `shohi_kani`: 消費税簡易課税
